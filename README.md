PeakThis
========

Highly customizable cross-platform peak fitting software

Maintainer
----------

Clemens Prescher (clemens.prescher@gmail.com)

Requirements
------------

    * python 2.7/3.4
    * PyQt4
    * numpy
    * scipy
    * lmfit
    * pyqtgraph
    
 It is known tu run on Windows, Mac and Linux.
